Unified Modeling Language (UML) adalah bahasa pemodelan perangkat lunak yang telah diakui sebagai standar untuk merancang cetak biru perangkat lunak. Selain itu, UML digunakan untuk tujuan visualisasi, spesifikasi, konstruksi, dan dokumentasi komponen-komponen dalam sistem  perangkat  lunak.Use  Case  Diagram adalah  salah  satu  jenis  diagram  UML  yang menggambarkan interaksi antara pengguna sistem dan sistem itu sendiri. Keterangan  pada use  case  diagram diatas  yaitu  saat user hendak  menggunakan aplikasi reminder, user diharuskan untuk memilih Login lebih dahulu,setelah itu user dapat menggunakan fitur yang ada pada aplikasi mulai dari jadwal kuliah, daftar tugas, kalender, notifikasi, akun dan logout.
